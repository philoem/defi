# DeFi-NFT

Dans ce repository vous trouverez tout d'abord un rush pour créer votre propre collection NFT.\
Par la suite vous allez découvrir la DeFi à travers la réalisation d'un deuxième rush pour créer un DEX simplifié avec votre propre oracle.

## Compétences

Au cours de cette partie, vous allez acquérir les compétences suivantes :

- Utiliser des tokens ERC20
- Utiliser des tokens ERC721
- Comprendre le fonctionnement d'un DEX
- Comprendre le fonctionnement d'un oracle
- Comprendre le fonctionnement d'un marketplace
